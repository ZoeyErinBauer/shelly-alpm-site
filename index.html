Arch Package Management, Reimagined.

Shelly is a modern, visual, and intuitive interface for your Arch Linux system. Not just a wrapper, but a native ground-up experience.

Get Shelly GitHub Repo

Quick Install

Run this command in your terminal to install the latest version of Shelly:

curl -fsSL https://raw.githubusercontent.com/ZoeyErinBauer/Shelly-ALPM/main/web-install.sh | sudo bash

Why Shelly?

Native Performance

Directly interacts with libalpm for the fastest and most accurate package management possible.

Modern UI

Built with Avalonia UI, providing a beautiful, responsive, and cross-platform ready interface.

AUR & Flatpak

Integrated support for Arch User Repository and Flatpaks, managing all your software in one place.
