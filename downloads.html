Web Installer (Recommended)

The easiest way to install Shelly on any Arch-based system.

curl -fsSL https://raw.githubusercontent.com/ZoeyErinBauer/Shelly-ALPM/main/web-install.sh | sudo bash

This script downloads the latest release and sets up both the UI and CLI tools.

Build from Source (PKGBUILD)

Standard Arch Linux way to build and install.

git clone https://github.com/ZoeyErinBauer/Shelly-ALPM.git
cd Shelly-ALPM
makepkg -si
